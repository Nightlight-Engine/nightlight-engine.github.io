About Nightlight

Nightlight is a game engine based on UPBGE from Zohan Haque which is in development and is not compiled yet. Nightlight is used to make custom games and, using custom lighting, you can even change the sun textures and skybox. You can model directly in Nightlight without using another 3D modeling software such as Blender or SketchUp. Nightlight also allows scripting in Python for your game instead of using a complex language such as C++. It comes with preinstalled plugins (not finished). Since Nightlight is really not complete, it is very complex to learn, as UPBGE is based on Blender which has complex tutorials.
